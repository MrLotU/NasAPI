## Requirements
iOS 8.0 or newer

## Installation

NasAPI-Swift is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following to your Podfile:

```ruby
platform :ios, '8.0'
use_frameworks!

target 'MyApp' do
pod 'NasAPI-Swift'
end
```
